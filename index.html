Try / Catch

Repo     ||     Deployed

Collaborative error-handling app for aspiring programmers

React frontend, Ruby / Sinatra backend, SQL database. Create issues that describe a problem, invite other users to work with you, and choose which issues you want to share with your collaborators. Users can post notes offering tips and guidance to issues they own or that have been shared with them. Users have full CRUD on issues that they created.



superrad

Repo     ||     Deployed

Image adventure game / colorful time-waster

Collaborative project with Peter Murray and Dakotah Ducharme. React frontend, Ruby / Sinatra backend, SQL database. The backend serves as an API for superrad, but it is also a fullstack app, Image Tagger, that we used to crowdsource 1.) seeding data for an image database, and 2.) adding one-word "tags" to the images. We used the data gathered thereby to create superrad, a relational image game that generates a layout of four images. Clicking an image generates a new layout based on images with related tags â€” with some randomness thrown in, to keep things interesting. Users who create an account can keep track of their current "image journey," or reset their path to start a new adventure.



Battle Doodles

Repo     ||     Deployed

Challenging reflex-based game for two players

Old school "two-players, one-terminal" browser game built in JavaScript (plus the Velocity.js animation library). After choosing three doodles as their champions from a pool of 10, the players square off in the arena, attempting to land blows and block their opponent's attacks based on careful timing. The doodles were lovingly hand-drawn with lo-fi, MSPaint-like freeware, and each doodle has a unique special ability to aid them in combat.



CardPool

Repo     ||     Deployed

Magic: The Gathering deckbuilder app

React frontend, Express backend, MongoDB / Mongoose data storage. Powered by the excellent Magic: The Gathering API. All users can search and view cards. Users who create an account can add to their deck any cards found through the search functionality. Users have full CRUD on decks they create.
